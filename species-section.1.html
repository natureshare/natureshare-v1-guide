The Species Section

When this page was last edited, NatureShare had over 16,000 species names in its database. This will continually rise to include all the species known to occur in Victoria.

The following items appear on the first page of Species Section:
  • Search (also accessed via the 'Quick Search' link on the main menu)
    The search box on the Species page allows you to search for a specific species, either by scientific name (eg. Acacia pycnantha) or common name (eg. Golden Wattle). The search operates by matching the text string as you type; eg. by the time you have typed "kook" there are only two remaining options in the drop down box (including Kookaburra).

    Quick tip    : The last three letters of the genus name and the first three letters of the species name is usually unique (eg. "cia pyc" for Acacia pycnantha). This is particularly useful for navigating through a genus like acacia or eucalyptus which tend to overwhelm the drop-down box because there are hundreds of species in the genus. The same applies whether it is the scientific name or the common name; eg. "den wat" for Golden Wattle also works well.

    Once you have chosen the species, press enter and you are taken to that species' page - see right for information about the Species page.

  • Scientific and Common Names
    Scientific names and many common names are already keyed into the database. No typos are possible when entering scientific names in observations or at any point in NatureShare.

    NatureShare may come up with a number of options for subsp./var./f. or s.l./s.s., etc. See the 'Which Species Name To Choose?' section for more details.

    If a common name is missing, you can add the common name by going to that species' page. If you see a common name misspelt you can change it by clicking on the 'Show deleting' function - see right.

    If there is more than one species with the same common name (eg. the Bright-eyed Brown butterfly) it will list them all in the dropdown menu and you wont know which one to choose. To bypass this problem you can either a) choose them one by one and see which is the one that you want, or b) go to the Advanced Species Search section and search for that common name (this will give you all the results for that common name on the same page for comparison). Wherever possible, common name duplication is avoided but it isn't always possible.

  • Advanced Species Search
    This allows you to search for groups of species (eg. Wattles of the Victorian Volcanic Plains) or to assist in identifying a species using information you know about that species; eg. by including (or excluding) any combination of Species Name (eg. eucalyptus), Common Name, Classification (eg. "family: orchidaceae"), Category (eg. parrot, fern, EPBC- or FFG-listed species, etc), Biological Attribute (eg. "flower: yellow", "petals: 5") and/or Collection (eg. within a specific collection - like 'Native Plants of Barrm Birrm, Riddells Creek').

  • Categories
    On this page is a list of all the Categories used in NatureShare (so far). Categories are particularly useful in the Advanced Search Section. Categories group together all the wattles, EPBC-listed (protected) species, extinct species, weeds, ferns, bats, parrots, etc. On this page, you can simply click on a Category and see all the species in that Category.

In the Search box, if you type in a scientific name or common name and select that species, you will access the Species Page. Every Species page in NatureShare has its own web address so it can be easily linked to from an existing website.

The Species page goes to the 'Details' tab by default. Click on the 'Observations' tab to see all the observations that have been made for that species. Once in the 'Observations' section it displays thumbnail photos (if one was taken) and details of all observation made for that species by default ... from here you can click on the 'Photos' tab to see larger size photos - NB. this displays a random set of photos so, in your browser, you can hit the 'Reload Page' button to get a new random selection of photos. At any point you can click on any photo to go to that specific observation. Once in the observation page the photo is displayed in a reduced form (about 60kb) and you can either a) click on the photo to download the full-size photo; b) hover over the boxes beneath the photo to download the file in various sizes, the last box being full-size.

The 'Details' tab (at the top of the Species page) includes the following information for each species (NB. once entered, this information is shared across all collections and users across Victoria):
  • Random Photo Selection; a random selection of thumbnail-size photos that have been entered for this species - click on any photo to go to that observation. In your browser, you can hit the 'Reload Page' button to get a new random selection. Click on any photo to view that specific photo and observation. The photo in the observation page is displayed in a reduced form (about 60kb) and you can either a) click on the photo to download the full-size photo; b) hover over the boxes beneath the photo to download the file in various sizes, the last box being full-size.
  • Common Names; lists the common names for that species. Once logged in you can enter different common names by clicking on the 'Add a Common Name' link (NB. avoid duplicating common names that are already applied to other species if possible). If you see a common name misspelt you can change it by clicking on the 'Show deleting' function (see below).
  • Classification; Classifications are ONLY relevant to the Species and Advanced Search sections. Classifications are formal/scientific classifications; eg. the two main ones are arguably 'family' and 'order'. These take the form "family: orchidaceae", "order: lepidoptera", etc. Classifications for individual species are listed on the relevant species page (eg. see Helena Gum Moth). Once logged in you can enter different classifications by clicking on the 'Add a Classification' link. You can also delete/change a classification entry if you see a mistake.
  • Categories; Categories are groups of species sharing common attributes that are recognised by a commonly used name; eg. 'parrot', 'wattle', 'butterfly', 'weed', 'rare_threatened', 'fern', 'fauna_introduced', 'extinct_vic', 'EPBC-listed', etc. Categories are ONLY relevant to the the Species and Search sections. A full list of categories is available on the main Species page and relevant categories for individual species are listed on the relevant species page (eg. see the category entries for the Amethyst Hairstreak butterfly). Once logged in you can enter different categories by clicking on the 'Add a Category' link. You can also delete/change a category entry if you see a mistake.
  • Biological Attributes; Biological Attributes are important attributes of a particular species. They are ONLY relevant to the Species and Search sections. The format for biological attributes is "Attribute: Descriptor" (eg. wing: green) which is usually in the form "Noun: Adjective". Each part of the attributes is a complete text string (no spaces) so multiple words must be hyphenated (eg. "head-hair: black") or use the underscore (eg. "juvenile_leaf: grey"). When you start typing in an Attribute, a dropdown box appears that lists all the similar Attributes used in NatureShare to date (ordered by popularity - most used Attribute is listed first). You don't have to use these and you can simply ignore the dropdown box but it can be an advantage to use what is already commonly used. Choosing from the dropdown box also helps to avoid the introduction of typos. There are two main types of Biological Attributes: 1)
    Basic or main observable characteristics of the species such as "flower: yellow", "petals: 5", "leaf: prickly", "body: yellow" - these are attributes designed for the Advanced Search system. 2) Detailed features that help distinguish it from other similar species (or subspecies). These are mostly not designed to be searchable (eg. for Drooping Mistletoe, "centre-flower-of-3: sessile" or for the Chocolate Lily "flower: 1-per-axil"). More on Attributes.
  • Similar Species; a list of other species that are closely related to the one detailed on that species page (eg. Inland Pigface, Carpobrotus modestus)
  • Metadata; this is non-editable data about the information on this page - including when the page was first created, etc.
  • Show deleting; click on the 'Show deleting' text and a number of lines appear on the page that allow you to delete mistakes that have been made. More on changing/deleting mistakes.
  • Hide functions; click on the 'Hide functions' text and the page changes to show only the data for this species and disables the editing options.
HOW YOU CAN HELP: If you find a species without Categories listed, without Classifications listed, without Biological Attributes listed, without Similar Species, please enter them (especially when setting up a new 'collection' or when a new species is added to a collection because without this information the collection cannot be searched properly). If you want to create new Categories and Attributes, NatureShare allows this - anything is possible.
