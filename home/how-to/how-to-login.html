How to Login

Anyone can use NatureShare as a resource but you need to login before you can upload observations, join/start/add to collections, add/change species attributes, etc.

Before you login, if you haven't got it on your computer already, download the free browser Mozilla Firefox or Google Chrome now ... NatureShare is designed to work with Firefox or Chrome (or Safari). If you are using other browsers, some important aspects of NatureShare will not work.

As always, NatureShare uses a sharing system for its login procedure, the OpenID login system. This means if you have an account with participating organisations (eg. Google, Yahoo, Flickr, etc) you don't need a new login name for NatureShare, you just use the one you already have.

First step is to click on 'login' when in NatureShare. This brings up a list of the biggest OpenID providers - visit OpenID to see more providers. You now have two options:
  1. Click on the icon for a group you have an account with (eg. click on the Google icon if you already have a Google account - see image below). If you are already logged into Google in another window then you will automatically become logged into NatureShare, otherwise the login box for Google will come up and you will login to NatureShare via Google (or Yahoo, WordPress, etc). If you haven't got accounts for any of the icons listed, you find it all works easier if you simply create a Google account just for this purpose (it is free and very simple).
  2. If you don't want to do any other the above, then you will have to click on the MyOpenID.com link to create an account (the red arrow is pointing to the MyOpenID.com link in the diagram below.


    Once the OpenID process has been completed you can login via your OpenID account by clicking on the large 'myOpenID'
    icon on the NatureShare login screen (see right). This will then prompt you for your OpenID username, then it will take you to a page where you can enter the password, and after this you are automatically sent back to NatureShare.

In all cases, if you tick the 'keep me logged in' or 'stay logged in' box, you will rarely need to formally login to NatureShare.


Why is a login necessary? For data security. If we didn't, then anyone can access and delete any data and we would never know who was doing it. This way we can record who makes changes and block any unruly people. But don't worry, a log is kept of all actions in NatureShare so, in true wiki style, we can restore things that are changed at any time.
