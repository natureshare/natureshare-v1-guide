Plants

There are over 3000 plant species in Victoria. For any local search, more often than not, the colour of the flower along with 1-2 other key attributes will often be enough to narrow the search within a 'Collection' to a feasible number of species.

HOW YOU CAN HELP: If you find a plant species without basic attributes entered in NatureShare, please enter them (especially when setting up a new plant 'collection' or when a new plant is added to a collection because without these attributes the collection cannot be searched). Attributes presented here are not meant to be exhaustive - if you want to enter other attributes that you know will help, NatureShare allows this - anything is possible.

There are actually very few attributes that are required for an effective search within any local collection. Indeed, the main parameters are:
  • Flower colour (usually only the main flower colour is entered - unlike with moth & butterfly wings where all colours are entered)
  • Flower type (see below)
  • Number of petals (if required)
These parameters need to be entered for all plants (see 'how you can help' above). Any search based on these 2-3 parameters will usually work well (with the possible exception of when wattles and eucalypts appear in the search results, but these can be easily excluded from the search if they are not wanted - see example searches at the bottom of this page).

The main searchable attributes for plants that are currently being entered are:
  • Flower colour; eg. "flower: blue", "flower: yellow", "flower: white", "flower: purple", etc.
    Usually only the main flower colour is entered (unlike for moth/butterfly wings where all colours are entered). The reason for this is if secondary colours are entered then it will often clutter up simpler, primary flower colour searches (eg. if "flower: white" and "flower: blue" were entered for "Isotoma fluviatilis" (pictured right) then it would be found under a search for "flower: blue" when the flower is obviously mostly white). If the flower is multi-coloured, comes in different colour forms or the colour could be interpreted in more than one way (eg. yellow, orange) then it is often best to split this information into separate attributes; eg. if a flower colour varies between yellow and orange, rather than one attribute, "flower: yellow-orange", searching often works better if two attributes are added - ie. "flower: yellow" and "flower: orange".
  • Flower type; the main options are:
    • "flower: petalled"
    • "flower: cluster"
    • "flower: tubular"
    • "flower: pea"
    • "flower: orchid"
    • "flower: grass"
  • Number of petals; (if applicable - usually only required for plants with the "flower: petalled" attribute)
    • options are: "petals: 1", "petals: 2", "petals: 3", "petals: 4", "petals: 5", "petals: 6", "petals: many"- on occasion "petal: 10" is used
  • Fruit colour; as above for flower colour (eg. "fruit: red")
  • Fruit type; options currently in use are "fruit: fleshy", "fruit: woody" and "fruit: nut"
  • Leaf characters; any distinguishing feature of the leaf (NB. many species may have no leaf characters listed, only deviations from the norm are entered)
    • "leaf: prickly"
    • "leaf: divided", "leaf: lobed"
    • "leaf: trifoliate"
    • "leaflets: 3:, "leaflets: 5", etc
    • "leaf: bipinnate"
    • "leaf: whorled" or more specifically how many leaves in the whorl if consistent (eg. "leaf: 4:whorled")
    • "leaf: serrated", "leaf: toothed"
    • "leaf: grey"
    • "leaf: hairy", "leaf: glabrous"; "leaf: rough"
    • "leaf: aromatic"
    • "leaf: strappy"
    • "leaf: tufted"
    • "leaf: linear"
  • Habit; the main options are:
    • "habit: tree", "habit: small-tree" (NB. a search just using the term "tree" will catch all)
    • "habit: shrub", "habit: low-shrub", "habit: large-shrub" (NB. a search just using the term "shrub" will catch all)
    • "habit: herb"
    • "habit: groundcover"
    • "habit: climber", "habit: climber-twiner" (NB. a search just using the term "climber" will catch all)
    • "habit: trailing"
    • "habit: spreading" (eg. suckering)
    • "habit: grass"
    • "habit: tufted"
    • "habit: mistletoe"
    • "habit: parasitic"
  • Stem characters; mostly "stem: spines", "stem: hairy", "stem: glabrous"
  • Bract colour; eg. "bract: yellow" (mostly for everlasting daisies)
Despite this long list, there are actually very few attributes that are required for an effective search within any local collection. Indeed, the main parameters are:
  • Flower colour
  • Flower type
  • Number of petals (if required)
Any search based on these 2-3 parameters will usually work well (with the possible exception of when wattles and eucalypts appear in the search results, but these can be easily excluded from the search if not required (see example for trees below).

Examples of searches that can be done with attributes entered for the plants in the Barrm Birrm collection (about 230 species):
  • Yellow with five petals
  • Purple pea flowers(as in picture top right of this page)
  • Pink tubular flowers (as in picture right) and again with prickly foliage
  • Trees (and again but excluding wattles and eucalypts)

HOW YOU CAN HELP: If you find a plant species without these basic attributes entered in NatureShare, please enter them (especially when setting up a new plant 'collection' or when a new plant is added to a collection because without these attributes the collection cannot be searched). Attributes presented here are not meant to be exhaustive - if you want to enter other attributes that you know will help, NatureShare allows this - anything is possible.
