Butterflies

A Tag is any important feature of the observation/photo (eg. male, mating, caterpillar).

Tags are ONLY relevant to the Observations section. When you upload an observation make sure you think about adding a Tag because Tags help build usable information about individual species. ... more info on Tags.

The following are examples of useful Tags for butterfly species:
  • Underside (if the photo shows the underside well) - see picture right
  • Upperside (if the photo shows the upperside well) - see picture right
  • Male
  • Female
  • Mating
  • Feeding
  • Worn (if it is photo of an old, worn individual)
  • ID (if the observation contains photo(s) or a compilation that shows the species ID features)
  • Caterpillar (NB. if you see a caterpillar of a butterfly or moth species it is very important to add the tag "Caterpillar" to your observation - without the "caterpillar" Tag, dates on caterpillar observations will get entered into the calendar for the flying time of the species - this problems arises because the caterpillar and adults have the same species name).
