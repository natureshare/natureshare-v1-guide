Plants

A Tag is any important feature of the observation/photo (eg. flower, fruit, leaf, male). If the plant you observed is in flower, it is very important to add the Tag 'flower' to your observation (see below).

Tags are ONLY relevant to the Observations section. When you upload an observation make sure you think about adding a Tag because Tags help build usable information about individual species. ... more info on Tags.

The following are examples of useful Tags for plant species:
  • Flower (NB. if the plant observed is in flower, entering a 'flower' tag is very important, otherwise your observation will not be entered into a flowering 'calendar' for that species)
  • An unusual colour form - eg. "white" if it is a white-flowered form of Hardendergia violacea, Wahlenbergia stricta, etc
  • Leaf
  • Fruit
  • Seed, Seedpod, Seedhead
  • Buds
  • Sepal
  • Trunk
  • Bark
  • Male - see example of Wurmbea latifolia (Broad-leaf Early Nancy) pictured right
  • Female
  • Habit
  • Form
  • ID (if the observation contains photo(s) or a compilation that specifically shows the species ID features)
For Eucalyptus species (and some other species) there might be other features that are helpful in identification; eg. for Eucalyptus species:
  • Juvenile_leaf
  • Adult_leaf
